# Feature Repo Template

This template is a text-only blueprint for extracting a feature into its own repository.

## Usage
1. Copy this folder into a new repo.
2. Replace placeholders in `package.json` and `src/feature.tsx`.
3. Publish or reference via Git dependency.

## Integration
- Import the `feature` export into `apps/sport_scale_app/src/features/registry.ts`.
- Ensure peerDependencies match the shell app.
